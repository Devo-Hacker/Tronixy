**Tronixy — Electronics E-Commerce Store**


 **About**

**Tronixy** is a full-stack e-commerce web application focused on electronics and tech accessories. It features a **Rule-Based Recommendation System** that helps users discover relevant products based on browsing and purchase behaviour. The platform provides a seamless shopping experience with product listings, cart management, user authentication, and order tracking.

**Features**

-**Product Search & Filtering** — Search and filter electronics by category
-**Rule-Based Recommendation System** — Smart product suggestions tailored to user behaviour
-**Shopping Cart** — Add, remove, and manage items before checkout
-**Order Tracking** — View and manage your past orders via My Orders page
-**User Authentication** — Register and log in securely
-**Responsive Design** — Optimised for both desktop and mobile devices
-**Fast Delivery & Trusted Service** — Built with quality and speed in mind

**Getting Started**

**Prerequisites**

- [Node.js](https://nodejs.org/) (v16 or above)
- npm

**Installation**

1. **Clone the repository**
   ```bash
   git clone https://github.com/Devo-Hacker/Tronixy.git
   cd Tronixy
   ```

2. **Install server dependencies**
   ```bash
   cd server
   npm install
   ```

3. **Install client dependencies** *(if applicable)*
   ```bash
   cd ../client
   npm install
   ```

4. **Start the server**
   ```bash
   cd ../server
   npm start
   ```

5. **Open the app**
   Open `client/pages/home.html` in your browser, or visit the live deployment at [tronixy.vercel.app](https://tronixy.vercel.app).
